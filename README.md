## chapter 03 - Laying the foundation

To run or build a project find scripts in package.json

```

"start": "parcel index.html",
"build": "parcel build index.html",
"test": "jest"
},
```

### `npm run start | npm start`

    Runs the app in the development mode

### `npm run build`
